Insurance Claims Analysis

Project Overview

Objective: To analyze claim frequency, severity, and payout ratios to identify high-risk customer segments.

Role: Power BI Developer

Data Processing (ETL)

The dataset contained policy details and claim transaction logs. Using Power Query, I:

  • Merged "Claims" and "Policy Holder" tables using a Left Outer Join.
  • Replaced null values in the "Settlement Amount" column with 0 for open claims.
  • Created age groups (bins) to analyze risk by demographic.

Technical Skills (DAX)

Key financial metrics calculated:

1. Loss Ratio (%)

Loss Ratio = DIVIDE(SUM('Claims'[Total Payout]), SUM('Policies'[Total Premium]), 0)

2. Claim Frequency

Claim Frequency = COUNTROWS('Claims') / DISTINCTCOUNT('Policies'[PolicyID])

Insights

  • Insight 1: Vehicle insurance claims spike by 15% during winter months.
  • Insight 2: The 18-25 age demographic has the highest frequency of claims but lower average severity compared to older demographics.
