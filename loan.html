Bank Loan BI Analysis

Project Overview

Objective: To analyze loan data and identify key risk factors contributing to defaults.

Role: Power BI Developer & Data Analyst

Data Processing (ETL)

The data was sourced from [Source Name]. Using Power Query, I performed the following transformation steps:

  • Removed duplicate rows to ensure data integrity.
  • Standardized date formats for time-intelligence calculations.
  • Created conditional columns to categorize loan grades.

Data Model

I utilized a Star Schema to optimize performance. The model consists of one fact table (Loans) and dimension tables for Customers, Locations, and Time.

Technical Skills (DAX)

Key measures created for this report:

1. Loan Approval Rate

Approval Rate = DIVIDE(COUNTROWS(FILTER('Loans', 'Loans'[Status]="Approved")), COUNTROWS('Loans'), 0)

2. Total Default Amount

Default Amount = CALCULATE(SUM('Loans'[Amount]), 'Loans'[Status] = "Defaulted")
